# 프로젝트 소개
간단한 쇼핑몰입니다.

 **기능**
 1. 로그인
 2. 회원가입
 3. 글작성/수정/삭제/상세보기
 4. 모든글보기/내피드보기
 5. 팔로우/언팔로우

## 제작 정보
vue-router 템플릿을 기반으로 만든 간단한 SNS사이트 입니다.
CodePresso 의  Dev-ops 과정을 수강하면 개발한 학습용 프로젝트 입니다.

## 사용 안내
CNS-vue 는 기본 포트를 8081로 설정해 놓았습니다.
CNS-boot 가 8080포트라는 전제하에 작성되 코드 입니다.

## 사용기술
 - HTML/CSS
 - Bootstrap Vue
 - Vue.js  :  @vue/cli 4.2.3
 - Node.js : v12.16.1.

## Dependencies

 1. Axios
 2. Bootstrap
 3. core-js
 4. vue
 5. vue-cookies
 6. vue-router
 
## 실행방법
 1. github에서 프로젝트를 다운받는다.
 2. 터미널에서 다운받은 dir로 들어간다. ( cd cns-vue )
 3. npm install
 4. npm serve
 만약 vue-cookie, vue-router, Axios 라이브러리 설치에서 문제가 생기면 별도록 설치를 하면 된다.
